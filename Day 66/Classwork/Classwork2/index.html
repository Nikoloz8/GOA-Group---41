HTML

HTML (HyperText Markup Language) - გამოიყენება ვებსაიტების სტრუქტურის შესაქმნელად. იგი ასახავს ვებსაიტების შინაარსსა და მასალას, როგორიცაა ტექსტი, სურათები, ხმარების ღილაკები და სხვა ელემენტები.

CSS

CSS (Cascading Style Sheets) - გამოიყენება ვებსაიტების ვიზუალური სტილის განსაზღვრისთვის. CSS საშუალებას აძლევს დეველოპერებს და დიზაინერებს გამოავლინონ HTML დოკუმენტების მახასიათებლები, როგორიცაა ფერები, შრიფტები, მანძილები და განლაგება.

JavaScript

JavaScript არის პროგრამირების ენა, რომელიც გამოიყენება ვებსაიტებისთვის ფუნქციონალის მისაცემად. იგი საშუალებას აძლევს დეველოპერებს, რომ დაამატონ ინტერაქტიულობა, რაც აუმჯობესებს მომხმარებლის გამოცდილებას.
