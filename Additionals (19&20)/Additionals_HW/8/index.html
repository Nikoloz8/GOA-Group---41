HTML&CSS ში არსებობს css ის დამატების 3 ძირითადი გზა. ესენია internal css, external css, inline css. inline css იწერება html დოკუმენტში მოცემულ tag ებში style ატრიბუტის საშუალებით. internal css ასევე იწერება html დოკუმენტში, თუმცა ამ შემთხვევაში ვიყენებთ style თეგს, რომელიც იწერება html-ის head-ში. external css ისთვის კი იქმნება ცალკე css გაფართოების ფაილი, რომელშიც იწერება ელემენტებისთვის, id-ებისთვის, class-ებისთვის განკუთვნილი property-ები. მისი დაკავშირება html დოკუმენტთან ხდება link თეგის დახმარებით.
